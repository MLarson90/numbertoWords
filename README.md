## _Numbers to words_
### _by: Max Larson_

### _Specs_

* _this program will return any value 1-20 with its name_
  _input: 11_
  _output: eleven_
* _this program will return a two digit number ending in 0 with its name_
  _input: 40_
  _output: forty_
* _this program will return any value 21-29 with "twenty" + Number_
  _input: 21_
  _output: twenty one_
